# C++ Garbage Collector
Пользователю предоставляется класс `GcPtr<T>`, который является оберткой над указателем. Чтобы получить его экземпляр, нужно вызвать функцию `GcNew<T>(Args...)` или же `GcNewArr<T>(size)`, если нужно получить массив.

Чтобы очистить мусор, нужно применить `GarbageCollector::GetInstance().CollectGarbage()`.

Также в программе запрещается брать адреса внутри выделенных областей, чтобы не нарушить работу программы.

### Сборка тестов
```
mkdir build
cd build
cmake ..
make BaseTest
./BaseTest
```
